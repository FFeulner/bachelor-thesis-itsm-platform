Dieses Projekt implementiert die Authentifizierung und Autorisierung via Zitadel (OIDC) für eine FastAPI-Anwendung.

## Voraussetzungen

- Python 3.10 oder höher
- Git
- Eine Zitadel OIDC-Instanz (Domain, Token-URL, JWKS-URL, Client ID, Redirect URI)
- `settings.env` im Wurzelverzeichnis: siehe unten
